# LikedIt Privacy Policy

This repository hosts the public privacy policy for the **LikedIt** browser extension.

LikedIt is a Chrome extension that highlights Instagram posts a user has already liked while browsing Instagram.

The extension processes information locally in the browser to determine whether a post has been liked and visually marks those posts. LikedIt does **not collect, store, transmit, or share any personal data**.

The privacy policy is published via GitHub Pages to provide a permanent public URL required for Chrome Web Store listing.
